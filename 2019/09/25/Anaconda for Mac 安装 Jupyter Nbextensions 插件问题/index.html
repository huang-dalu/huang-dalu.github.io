Anaconda for Mac 安装 Jupyter Nbextensions 插件问题。

关于 Nbextensions

Nbextensions 是常用的 Jupyter 插件，包括代码补全、代码高亮、目录美化等多种功能，以下是 
Nbextensions 有的功能，非常齐全。

官方介绍：https://jupyter-contrib-nbextensions.readthedocs.io/en/latest/

 Nbextensions 安装

对应版本：

  • Mac 10.15.2
  • python 3.7.4
  • anaconda 1.9.7

常规安装流程：

正常情况下，会在 Jupyter 主页显示「nbextension」的菜单按钮，点击设置对应插件即可。有部分情况会不显示，可以分别采用以下解决方案。

方案一

  1. 打开 http://localhost:8888/nbextensions?nbextension=toc2/main
  2. 是否能找到页面，如果能找到页面，在页面设置即可，如果页面报错不存在，可以尝试方案二。

方案二
