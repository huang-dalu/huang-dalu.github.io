Python 中的 i++，i+

顺便在这篇基础下面总结下 Python 中的 i++，i+ 这类写法和 Java 不一样。在新手入门时会有点疑问，这里列一下。

i+1

这种都是一样的写法，没什么问题，很好理解。

i+

i+ 就是 Python 中的自增写法。相当于其他语言的 ++i。

i-

Python 中的自减写法。

++i

Python 中的 ++i，仅仅是作为判断运算符号,类似数学中的负负得正。

i++

Python 中不支持这类写法，所以直接报错。
