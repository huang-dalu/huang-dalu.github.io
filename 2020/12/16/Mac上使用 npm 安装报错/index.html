Mac上使用npm安装报错

2020-12-16

例如使用 npm 安装 Hexo

报错信息如下：

原因: 执行命令行时没有获得管理员权限
解决方案：在命令行前面添加 sudo 获取管理员权限，输入管理员密码就行
例如原来的是:  npm install -g hexo-cli
应该改为：sudo npm install -g hexo-cli

再输入密码就行。
