添加 SSH key 到 GitHub

点击 GitHub 个人资料的设置操作。侧边栏找到导航“SSH and GPG keys”

单击“ New SSH key”

在“Title”字段中添加描述性标签，Key 字段中粘贴复制的 id_rsa.pub 内容。最后点击“Add SSH key”。最后 GitHub 需要弹窗确认密码就行。
