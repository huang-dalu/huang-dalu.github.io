乐刻运动的业务梳理

2021-03-07

Contents

  1. 乐刻健身需求供给情况
  2. 乐刻健身服务
  3. 乐刻集市
  4. 乐刻社区
  5. 乐刻用户标签

楼下有两家乐刻健身门店。于是对乐刻的「月卡」模式有兴趣。所以周末花了点时间了解了下乐刻的一些业务和数据指标。现在分享下。

内容包括：乐刻需求-供给情况说明、乐刻健身服务业务指标、乐刻集市业务指标、乐刻社区业务指标、乐刻用户标签。

乐刻健身需求供给情况

这部分可以查看乐刻创始人的一篇分享。我这里用脑图整理了下。

乐刻健身服务

第 2 部分是乐刻提供的一些健身服务。

1. 内容

  • VIP会员（月/季度/年卡）
  • 团体课：显示近5天课程，可以提前约近3天；课程丰富
  • 私教（1对1）：每个教练服务4个左右健身房；在线约课；价格220.00/节
  • 训练营（1对多）：内容多（瑜伽、舞蹈、格斗）；常规1000.00/10节课。平均单价100.00/课

2. 关键指标
总用户LTV指标维度 = 乐刻VIP会员 + 团体课 + 私教课 + 训练营 + 乐刻集市订单金额 + 会员评测参与度 + 社区贡献度

用户参与度指标

  • 课程消耗率（不同用户/不同课程对比）。课程消耗率与续费率正相关；
  • 用户整体生命周期，用户贡献的课程数、销售额；
  • 用户在不同场地数据情况；
  • 参与频率；是否有老带新；
  • 新用户激活节点/老用户流失节点，以及特性。

课程/教练度量指标

  • 各个课程用户数比较；
  • 私教教练的学员数量对比；
  • 教练在线回复率、学员互动情况对比；
  • 内部各个地区场地对比：课程/教练是否饱和。

3. 想法

  • 是否考虑所有的课程（私教、团体课、训练营）都有对应的视频介绍。可以做 A/B Test 论证结果；
  • 是否会考虑一些课程直接做到线上？
  • 乐刻首页推荐「我的门店」，推荐逻辑是否只是按根据距离推荐？（是否有考虑优先推直营店）
  • 直营店和合伙人店，具体有哪些区别。比如：场地大小/课程/私教会有区别吗？
  • 平台会给场地分配教练员（不同时间），那教练员是否可以自主选择？

乐刻集市

第 2 部分是乐刻集市的业务指标。

1. 模式

  • 商品分类 ：运动补剂、代肉蛋白、低卡饮品、生鲜粗粮、服饰配件、道件护具、器械装备；
  • 拼团：10款不到商品拼团。是作为引流活动款；
  • 乐币抵扣：提升活跃用户的签到积分工具。目前支持 16 款商品抵扣部分；
  • 0元试用（独立小程序）：为会员评测提供评测内容数据；
  • 商品推荐：商品列表第 1 款固定为「鸡胸肉」引流款；固定商品推荐排序（刷新不会换一组商品）。

2. 关键指标
总用户LTV指标维度 = 乐刻VIP会员 + 团体课 + 私教课 + 训练营 + 乐刻集市订单金额 + 会员评测参与度 + 社区贡献度

全局指标

  • 在集市产生订单用户数与乐刻VIP用户数比率；
  • 分类指标：男女贡献订单数、销售额指标；穿戴器材与食品贡献订单数、销售额指标；
  • 用户整体生命周期，用户贡献的订单数、客单价；
  • 模式比较：用户在不同模式下产生的订单、销售额指标。比如：拼团和推荐商品模式比较；
  • 乐刻集市用户与乐刻会员等课程，是否存在正线性相关。

新用户指标：不同乐刻推广渠道的用户，是否对乐刻集市有影响（对比指标）。

老用户指标（以购买单数区分，比如：购买5单以上）：新老用户是否在集市购买商品类目有差异。比如：新用户买食品，老用户买器材。

3. 想法

  • 「商品推荐」目前是固定顺序，且没有区分新老、男女客户，导致推荐但部分商品不精准。是否可以考虑：新老客户 + 购买课程情况 + 男/女 + 浏览商品 + 订购商品 来划分权重做推荐；
  • 由于部分商品客单价高。建议在商品详情 + 订单详情页面，加上「联系客服」，提升转化率；
  • 会员评测内容数据。数据存在过期情况。比如「西冷牛排」已经下架。但是评测区首屏还是有多条相关商品评价。建议评测数据和热门商品对应。另外是评测内容格式，建议在产品后台有相关编辑和选项（如：正文/标题/口感描述/颜值描述等等）辅助用户填写，提升用户阅读；
  • 是否考虑让教练分销商品。教练在适当场景推荐，用户在购买商品时，填上教练名字/编号，教练有分成。目的提升销售额；
  • 是否考虑将商品按照健身目的作为组合套餐（比如：减脂套餐A+B+C组合）。用户达到健身目的，平台提升客单价；
  • 乐刻集市的商品，是供应商代发还是乐刻自己自营发货呢？

乐刻社区

乐刻和 Keep 一样，也有社区功能。

1.社区分类

  • 教练社团：以教练 + 教练学员为主的聚集地
  • 兴趣社团：以健身目的为兴趣的学员聚集地

2.关键指标
定义社区优质内容维度。用户关注：各类课程情况、教练情况；健身方法、效果；健身好友动态。平台关注：围绕课程、教练、健身、健身学员关系的内容。

用户参与度指标

  • 发帖数量、频率、质量（图文情况）；
  • 互动数据：点赞、评论、关注情况；
  • 加入社团个数。

想法

  • 是否可以将社区和乐刻集市的乐币做关联？比如：发帖/评论/点赞有对应乐币；
  • 乐刻App首页的「同城社团」目前是固定顺序（再次刷新依然是固定的），内容兴趣命中率不高。是否可以考虑：购买课程情况 + 男/女 + 加入的社团 + 关注好友 来划分权重做推荐。

乐刻用户标签

最后是根据乐刻的业务模式，整理乐刻的业务标签体系，这类标签用来做用户增长的支撑。用户画像、广告系统会用到。当然整理的只是作为局外人看到的内容。

以上，是关于乐刻的一些业务梳理。
