产品力的维度

2017-11-19

产品和需求之间，存在着一一对应的关系，“一一对应的程度”称之为产品力。对应程度越高，产品力越强。则生命周期越长。

产品力 = 基本价值 + 核心价值 + 附加价值 + 门槛系数

  • 基本价值：目标用户对产品核心属性的满意程度
  • 核心价值：目标用户对产品的认可程度
  • 附加价值：产品给用户带来附加价值时的惊喜程度
  • 门槛系数：竞品对产品模仿时的难易程度

例如:以“洗衣粉”进行研究。顾客对洗衣粉最基本的使用需求是其“去污能力”，对此可进行调查。

设定：顾客对“去污能力”程度的评价用分数来描述。即：核心属性 → 使用需求。设:满意度MO，取值： 0≤M0≤10，量纲：度

得：经调查该洗衣粉“去污能力”目标顾客评分为7度（假定值）。

推理：

  • RO （认可度）对洗衣粉包装或规格等的认可度，设定为 8 度。
  • XO （喜悦度）洗衣粉的清香味或其他将给顾客带来的额外惊喜，设定 X0=7 度
  • YO（门槛系数）设定 Y0=8 度

结论：MO + R0 + X0 + YO = 7 + 8 + 7 + 8 = 30度

以上公示来自传统行业，我稍微改良了下。应该还需要加上「基础价值」 「核心价值」两项的权重，必须要到 6 分

以上，才具有对比衡量的可能。
