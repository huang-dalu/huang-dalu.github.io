竞品分析小练习：设计一个果汁饮料，如何进行竞品分析？

2015-01-03

问题：设计一个果汁饮料，竞品有哪些，如何进行竞品分析？
设计一个果汁饮料，竞品有哪些，如何进行竞品分析？
精简问题：如何对果汁饮料（产品）进行竞品分析？
问题拆分：

  1. 竞品分析目的是什么？
  2. 怎样才算竞品？（寻找竞品范围界定）
  3. 分析内容是什么？（需要根据目标来定）
  4. 分析，得出结论

比较研究法：两个或两个以上事物的对比，以找出它们之间相似性与差异性的一种方法。

  1. 竞品分析目的是什么？
  2. 设定分析目标 —— 竞争对手
  3. 将同类功能模块或外观细节编组作表
  4. 根据比较结果作进一步分析，得出结论

回答：

问题 1：竞品分析的目的是什么？
问题拆分：
（1）我们产品（功能）的定位是？
（2）和竞品对比后我要得到的是？

回答（1）
假定我们需要设计果汁饮料的定位是：20-40 在减肥过程中的一款功能性饮料（也就是主打女性市场的减肥饮料），也就是说我们这次竞品分析的目的是从 0 到 1 做一款减肥饮料（单个分析道理一样）

回答（2）
竞品分析后，我需要有如下结论：

  • 饮料的颜色是（ UI）？
  • 饮料的口感是（交互）？
  • 饮料的含量是（功能模块）？
  • 产品的包装是？渠道？（运营推广策略）
  • 减肥饮料的市场趋势是？业界现状是？
  • 竞争对手的优缺点是？核心优势是？（SWOT）？
  • 竞争对手的的产品定位，发展规划和策略是（纵向研究竞争对手的功能侧重点，产品迭代方向）？
  • 整个行业趋势是向上还是向下？（市场数据）
  • 目标用户研究。心理、年龄、消费习惯等等（用户研究）

参考：
1.战略层（产品定位、用户需求)
2.范围层（主要功能）
3.结构层（信息架构）
4.框架层（交互设计）
5.表现层（视觉设计）
—— 来自《用户体验要素》

问题 2：怎样才算竞品？找竞品（寻找竞品范围界定）
回答：
根据需要的结论倒推需要研究哪些维度，再找出那些维度里市场反应不错的产品。

颜色

  • 我们能做出多少种颜色的饮料？
  • 市场上总共有多少种颜色的饮料？
  • 减肥类同行业大概有多少种颜色的饮料？
  • 用户喜欢什么颜色的饮料

口感

  • 市场上总共有多少种口感的饮料？
  • 减肥类同行业大概有多少种口感的饮料？
  • 用户会喜欢什么样的口感？

饮料的含量是（功能模块）？

  • 我们能怎么做？
  • 竞品的主要成分（功能模块）是？

包装、渠道

  • 市面的流行包装是？
  • 柱状物能走哪些包装？
  • 用户喜好？
  • 竞品的各类渠道？主打哪些地方？一线／二线城市，城镇，小区（分发市场和合作商）

查看行业报告，行业数据

回答 3：
分析内容，将以上的收集的信息，按照表格排列，分析，得出一些可以定量的数据。

回答 4：
根据分析目的，得出分析结论：
举例：

  • 饮料的颜色是（UI）？——黄色
  • 饮料的口感是（交互）？——微甜
  • 饮料的含量是（功能模块）？——xxxx
  • 产品的包装是？渠道？（运营推广策略）——主打一线城市
  • 减肥饮料的市场趋势是？业界现状是？——市场还未饱和，xxx
  • 竞争对手的优缺点是？核心优势是？（SWOT）？——xxxx
  • 竞争对手的的产品定位，发展规划和策略是（纵向研究竞争对手的功能侧重点，产品迭代方向）？——A主打中高端白领，B主打xxxx
  • 整个行业趋势是向上还是向下？（市场数据）——折线图
  • 目标用户研究。心理、年龄、消费习惯等等（用户研究）——20岁，白领xxxx

最后的最后，如果行业已经有了不可撼动的标杆，是不是可以想想，往减肥的方向深挖。中药？为什么减肥？不正常饮食？（当然这可以算在用户研究范畴）
